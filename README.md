📊 Customer Shopping Behavior Analysis
📌 Overview

This data analytics project focuses on analyzing customer shopping behavior to generate meaningful business insights. The project covers the complete analytics workflow — from data loading and exploration in Python, data cleaning, SQL analysis, to dashboard creation in Power BI and final reporting through PPT and documentation.

The goal is to understand revenue trends, customer segments, product performance, and purchasing patterns to support data-driven decision making.

📂 Dataset

Customer shopping transactions dataset

Includes features such as customer demographics, product categories, purchase amount, shipping type, subscription status, and ratings

Contains missing values that were handled during the data cleaning process

🛠️ Tools & Technologies

Python – Pandas, NumPy, Matplotlib / Seaborn (EDA & preprocessing)

SQL – PostgreSQL / MySQL / SQL Server (data querying & analysis)

Power BI – Interactive dashboard creation

Gamma – Presentation (PPT) design

GitHub – Project documentation & version control

🔎 Project Steps
1️⃣ Data Loading

Imported dataset using Pandas

Checked structure, data types, and summary statistics

2️⃣ Exploratory Data Analysis (EDA)

Analyzed revenue distribution

Studied customer demographics and purchasing patterns

Identified top-selling and highly rated products

3️⃣ Data Cleaning & Transformation

Handled missing values

Renamed columns and standardized formats

Created new features such as age groups and customer segments

4️⃣ SQL Analysis

Loaded cleaned dataset into relational database

Wrote SQL queries to analyze revenue trends, discounts, subscriptions, and shipping performance

5️⃣ Dashboard Development

Built an interactive Power BI dashboard

Visualized KPIs such as total revenue, customer segments, product performance, and sales trends

6️⃣ Reporting & Presentation

Created a business insights report

Designed a professional presentation using Gamma

📊 Dashboard Highlights

Revenue analysis by gender and age group

Product category performance

Customer segmentation (new, returning, loyal)

Discount and shipping impact on sales

✅ Key Results

Identified high-revenue customer segments

Found top-performing products and categories

Observed impact of discounts and subscription status on purchase behavior

Generated actionable recommendations for marketing and customer retention
